# Thumblify 🚀

Build an AI Thumbnail Generator App using React | MERN Stack + Pollinations AI API, Gemini API

## 🌐 Live Demo

Frontend: https://thumblify-virid.vercel.app  
Backend: https://thumblify-server-drab.vercel.app

---

## ✨ Features

- AI Thumbnail Generation
- User Authentication
- Login & Signup
- Save User Thumbnails
- Responsive UI
- MongoDB Database
- Cloud Image Storage
- Session Authentication

---

## 🛠️ Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS
- Axios
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Express Session
- bcryptjs

---

## 📦 Installation

### Clone Repository

```bash
git clone https://github.com/balajigs08/thumblify.git
```

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

### Backend Setup

```bash
cd server
npm install
npm run server
```

---

## 🔐 Environment Variables

Create `.env` file inside `server` folder.

```env
MONGODB_URI=your_mongodb_uri
SESSION_SECRET=your_secret
CLOUDINARY_CLOUD_NAME=your_cloud_name
CLOUDINARY_API_KEY=your_api_key
CLOUDINARY_API_SECRET=your_api_secret
```
